# 更新日志 (CHANGELOG)

所有重要的更改都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

---

## [1.3.1] - 2026-04-10

### 新增
- 调用日志列表（包括进行中请求）显示请求的 host
- 浮动窗口支持滚轮上下滑动
- 工具调用浮窗内容超出时自动滚动到底部

---

## [1.3.0] - 2026-04-10

### 新增
- 版本号更新至 v1.3.0

---

## [1.2.1] - 2026-04-10

### 新增
- 浮动窗口支持工具调用详情面板
  - 主浮窗右侧自动弹出工具详情浮窗，实时显示工具名称和输入参数
  - 参数随 SSE 流式增量更新
- 浮动窗口底部新增实时 token 统计（in / out）

### 修复
- 修复右侧工具详情浮窗布局截断问题，卡片占满窗口且支持滚动

---

## [1.2.0] - 2026-04-10

### 新增
- 调用日志详情面板支持自由拖动调整宽度
- 请求体、响应体、汇总内容支持上下拖动调整高度
- 面板大小记忆：切换不同请求时保持自定义的大小不变

### 修复
- 修复请求详情面板内滚动时背景色丢失的问题
- 修复详情面板内容区域上下拖拽不跟手、突然断开的问题

### 优化
- 实时请求原始数据（SSE）区域也支持高度调整

---

## [1.0.1] - 2026-03-09

### 修复
- 添加 `resources/icon.png` 修复 README 图标显示

## [1.0.0] - 2026-03-08

### 新增
- 初始版本发布
- 平台管理功能
  - 支持添加多个 AI 平台配置
  - 支持 OpenAI 协议和 Anthropic Claude 协议
  - 自定义 API Base URL
  - 单端口多路径代理架构
- 本地代理服务
  - 一键启动/停止代理
  - 自动处理 API 认证头
  - 支持 SSE 流式响应
  - 请求/响应自动记录
- 实时监控
  - 每个平台独立的输出窗口
  - 实时显示流式响应内容
  - Token 统计功能
  - 首 Token 延迟统计
  - 输出速度 (tokens/s) 监控
- 日志系统
  - 完整的请求/响应记录
  - 流式数据汇总
  - 支持按平台筛选
  - 日志导出（JSON/CSV）
  - 工具调用 (tool_calls) 汇总记录
- 设置中心
  - 主题切换（浅色/深色/跟随系统）
  - 日志保留策略
  - 代理端口配置
  - 开机自启动
- 浮动窗口
  - 桌面歌词式输出窗口
  - 实时显示 AI 输出内容
  - 支持文本、思考、工具调用等内容类型
- 自动发布
  - GitHub Actions 自动构建
  - 支持 macOS 和 Windows
  - 自动创建 Release
