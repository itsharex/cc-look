# Resources

This directory contains application resources such as icons and images.

## Required Files

- `icon.png` - Application icon (512x512 recommended)
- `icon.ico` - Windows icon (256x256 recommended)
- `icon.icns` - macOS icon

## Creating Icons

You can use tools like:
- [electron-icon-builder](https://www.npmjs.com/package/electron-icon-builder)
- [icon-gen](https://www.npmjs.com/package/icon-gen)

Or manually create icons using image editing software.
